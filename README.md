# Wonderly Landing Page (GitHub Pages)

## Quick start
Upload these files to a GitHub repo (root):
- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.png`

## Deploy to GitHub Pages
GitHub → Repo → **Settings → Pages**
- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/** (root)

## IMPORTANT: Disable Jekyll
Add an empty file named `.nojekyll` in the repo root.

## Update waitlist email
Open `script.js` and replace:
- `youremail@example.com`
with your real email address.
